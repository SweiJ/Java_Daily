# 【项目实践】使用 MinIO 自建高性能对象存储服务器

## 项目运行

本项目使用 Spring Boot 搭建，将项目导入到 IDE 即可运行，默认端口 8080。

> 注意，要提前在服务器搭建好 MinIO
